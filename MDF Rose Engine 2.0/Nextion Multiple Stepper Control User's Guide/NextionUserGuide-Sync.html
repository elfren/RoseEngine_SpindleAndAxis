Synchronization Screen

Purpose: Synchronizes the Spindle and an Auxiliary Axis. This is used to

  • produce fluting on the cylindrical surface of workpiece (e.g., using a curvilinear slide),

  • produce helical patterns on the cylindrical surface of workpiece,

  • produce threads on the cylindrical surface of workpiece, or

  • produce spirals on a face surface.

There is value in reading thru the threading examples below as they can be helpful for understanding how to add helixes to an object.

Helix Type The Helix Type setting, Left or Right, correspond to the idea of threading (I.e., left-handed or right-handed threads). The same thinking should be applied to how you have the helix cut.
 
Variables
  Distance   The value of the Distance variable determines the cut length (amount of Z axis movement) and is dependent upon value entered in Configuration Screen for Distance/360. (The default for this on the MDF Rose Engine 2.0 is 10.)

To change this value, touch the number, and you will be presented with the Number Pad Screen.
 
  Revolutions   The value of the Revolutions variable determines how far around the piece the helix will travel based on the spindle rotating. The value of 6 illustrated will rotate the spindle \[ \begin{align} Travel \frac{\mathrm{deg}}{\mathrm{rev}} & = 360 \, \mathrm{deg} \times \, Revolutions \\ & = 360 \, \mathrm{deg} \times \, 6 \\ & = 2160 \, \mathrm{deg} \end{align} \] To change this value, touch the number, and you will be presented with the Number Pad Screen. This is used when using a Spherical Slide, and is a rotation about the Y axis.

Right and left speed bars (for Spindle and Axis, respectively, are shown as a percentage of maximum speeds set on the Configuration Screen for Sync and Z Axis, respectively. Note that slowest value on speed bars will control the other value.

When cutting with a spinning cutter, just as on a router, it is important to always cut downhill. With the cutting frame on a rose engine lathe, this leaves the smoothest surface.

The choice of direction for the Z axis should be made so that the cutter is advancing into the direction to make that happen. Choose the Z axis direction (red arrow) so that cutting (blue arrow) is downhill.

This is an advantage for using a fly cutter over a carbide bit. With a fly cutter, the direction of the cut can be reversed. (For more information about fly cutters vs. carbide bits, see also Cutters on the Cutting Frames page at the Ornamental Turning Book of Knowledge.)

Thread Cutting for Boxes with the Synchronization Screen

Note: refer to instructions in other resources such as Turning Threaded Boxes by John Swanson, Schiffer Publishing, 1999 for box threading tips and techniques.

Example 1: Single-start thread of 16 threads per inch (1/16 inch pitch)

  1. Set Revolutions to the number of threads to be cut around the cylinder. Suggest using 4 as a starting point.

  2. Set Distance to .25 (four threads at 1/16 inch = 1/4 inch).

  3. Setup a 60-degree double angle cutter running in a drilling frame, and the drilling frame set parallel to the lathe's spindle (driving by the overhead drive is recommended). Test and adjust the Spindle and Axis speed bars as necessary to get a usable combination of feed and speed (not so fast as to bog down cutter, not so slow as to burn the wood). If necessary, adjust the MaxSpd setting on the Configuration Screen to get a usable combination.

  4. Test Left and Right radio buttons to determine which type of thread will be cut (left hand or right hand).

  5. Choose whether to cut In or Out based on what makes sense for your piece.

  6. Cut the threads. A typical good practice uses roughing and final cuts for each thread. If the wood tends to fuzz up or chip out, it is a common practice to flow some CA glue into the rough threads and let it cure prior to the final cut.

Example 2: Four-start thread of 16 threads per inch (1/16 inch pitch, 1/4 inch lead)

  1. Set Revolutions to the number of threads (of each start) to be cut around the cylinder. Suggest using 1 as a starting point.

  2. Set the Distance to .25 (1/4 inch lead). This will cut one continuous thread traveling 1/4 inch as it goes once around cylinder.

  3. Got to Index #1 Screen. Select Div toggle and set value in Size box to 4. Note: would also work to select Deg toggle and set value in Size box to 90.

  4. Follow steps 3, 4, 5 & 6 in Example 1 above.

  5. Return to the starting point with cutter retracted from work using In or Out as appropriate.

  6. Index the spindle using either CW or CCW based on personal preference to get to starting point of next thread start. Re-engage cutter and repeat cutting process(es) as outline in step 6 in Example 1 for Start #2. Retract the cutter and return to start point of Start #2.

  7. Repeat indexing and cutting procedures for Starts #3 and #4.
